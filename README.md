Простейшее django-приложение.

1. Реализованы модели Product и Category.
2. Product - дочерняя модель, связана через ForeignKey с Category, родительской моделью.
3. Присутствует кастомная команда для генерации случайных тестовых данных в БД, для удобства в разработке
   (запуск с помощью python manage.py gen_data).
4. Созданы и применены миграции.
5. Настроил шаблоны и представления для отображения категорий и списка продукта.
6. Настроил формы для добавления и редактирования продукта.
7. Настроил отображения ошибок валидации.
8. Добавил кастомизацию: list_display, search_fields.
9. Написал кастомные действия через admin.action для изменения цены.
